# Alsat.Asia — Central Asia Financial Intelligence

Полнофункциональный многоязычный сайт финансовой аналитики по Центральной Азии.

## 📂 Файлы

| Файл | Назначение |
|------|-----------|
| `index.html` | Главный файл сайта (всё внутри — HTML+CSS+JS+данные) |
| `404.html` | Страница ошибки 404 |
| `robots.txt` | Инструкции для поисковых роботов |
| `sitemap.xml` | Карта сайта для Google/Bing |
| `rss.xml` | RSS лента со всеми статьями |
| `README.md` | Этот файл |

## ✨ Возможности

- **4 языка** — English / Español / Русский / Қазақша
- **10 готовых статей** с реальными данными 2026 года, на всех 4 языках
- **Переключатель темы** — светлая/тёмная
- **Поиск** по статьям
- **Адаптивный дизайн** — мобильный + десктоп
- **Сохранение настроек** — язык и тема в localStorage
- **SEO готов** — meta теги, Open Graph, Twitter Card, JSON-LD
- **Cookie banner** — GDPR совместимый
- **Favicon** — встроенный SVG (не требует файла)
- **Bigger CTA** — Newsletter форма для сбора email
- **Disclaimer** — на каждой статье и в footer (юридическая защита)

## 🚀 Локальный деплой

### Вариант 1 — Python (самый простой)
```bash
cd папка_с_файлами
python3 -m http.server 8000
```
Открыть `http://localhost:8000`

### Вариант 2 — Node.js
```bash
npx serve .
```

### Вариант 3 — PHP
```bash
php -S localhost:8000
```

## 🌐 Деплой на alsat.asia

Просто загрузите все файлы в корень сайта через FTP/SFTP или панель управления хостинга.

## 📝 Что добавить позже

После запуска YouTube канала:
- В JS файле найти `alert('YouTube channel coming soon')` и заменить на реальные ссылки
- Обновить URL канала в JSON-LD структурированных данных
- Добавить ссылки на соцсети в `og-image.png` и `sameAs` массив

## 🔧 Настройка под себя

### Изменить тексты статей
Найти `const ARTICLES = [` в `index.html` — там все 10 статей на 4 языках.

### Изменить UI переводы
Найти `const UI = {` — все надписи интерфейса на 4 языках.

### Изменить цвета
В CSS найти `:root` — основные переменные:
- `--gold` — акцентный цвет (золотой)
- `--paper` — фон
- `--ink` — текст
- `--rust` — красный для YouTube кнопки

## 📊 Бегущая строка с данными

# Проект alsat.asia — Контекст для Claude

## Что это
**alsat.asia** — казахстанский сайт бесплатных объявлений (классифайд). Аналог OLX/Kaspi объявлений, но бесплатный и без комиссий. Целевая аудитория: Казахстан, города Астана, Алматы, Шымкент, Бурабай и др.

Владелец: Адил Курманов (adilkurmanov9@gmail.com)

---

## Стек
- **Фронтенд**: один файл `index.html` — ванильный HTML + CSS + JavaScript, никаких фреймворков
- **База данных**: Supabase (REST API, вызывается прямо из JS в браузере)
- **Хостинг**: Vercel (auto-deploy из GitHub при каждом пуше в main)
- **Репозиторий**: `github.com/Adilveis81/burabay-website`
- **Локальный путь**: `/Users/adilkurmanov/burabay-website/index.html`

---

## Деплой
```bash
# Через osascript (не через bash — нет keychain в sandbox)
cd /Users/adilkurmanov/burabay-website
git add index.html
git commit -m "описание"
git push origin main
```
- Vercel подхватывает пуш и деплоит автоматически (~1 мин)
- **Лимит Vercel Free: 100 деплоев/день** — при частых правках можно упереться
- Если упёрлись: ждать сброса (~через сутки), потом `git commit --allow-empty -m 'trigger deploy' && git push`

## Дизайн и UI
- **Цветовая схема**: оранжевый (`#FF6B00`) — основной бренд, синий/зелёный акценты
- **CSS переменные**: `--o` (оранжевый), `--t` (бирюзовый), `--g` (зелёный), `--b` (синий)
- **3D pill кнопки**: box-shadow градиенты, используются везде (Войти/Выйти, Подать объявление и т.д.)
- **Адаптив**: CSS Grid с `data-cols` атрибутом на `<html>`, медиазапросы на 480/640/900/1200px
- **Иконки**: Tabler Icons (CDN)

## Ключевые блоки на странице (сверху вниз)
1. **Navbar** — логотип, ссылки (Объявления / Услуги / Вопросы / Поддержка), языковой дропдаун (RU/KZ/EN), кнопка Войти/Выйти
2. **Urgency bar** — бегущая строка: таймер акции, кол-во объявлений, 0₸ комиссия, кнопка "Подать объявление"
3. **Forex ticker** — бегущая строка с курсами USD/EUR/RUB/TRY/CNY к тенге (обновляется в 7:00/12:00/16:00 Алматы)
4. **Hero** — поиск, кнопка разместить объявление
5. **Категории** — 12 карточек с реальным количеством объявлений из Supabase
6. **Лента объявлений** — карточки из Supabase + заглушки
7. **Секция приложения** — кнопка Telegram бота (@Alsat_Asia_bot)
8. **FAQ** — 8 вопросов на 3 языках (ru/kz/en)
9. **Поддержка** — форма обратной связи

## Telegram бот (@Alsat_Asia_bot)
- Роль: модерация объявлений, проверка на спам/ошибки/безопасность, ответы на вопросы по публикации
- НЕ принимает заявки и не публикует объявления напрямую

## Бегущие строки — как работают
```css
@keyframes fxScroll { 0%{transform:translateX(0)} 100%{transform:translateX(-50%)} }
```
- Контент дублируется через `element.innerHTML += element.innerHTML` (флаг `dataset.duped`)
- Пауза при hover: `animation-play-state: paused`
- Forex: `open.er-api.com/v6/latest/USD` — бесплатный API с CORS

## Категории и Supabase
```javascript
// CC_MAP связывает id элемента с полем catLabel в БД
var CC_MAP = {
  'cc-auto': ['Авто'], 'cc-realty': ['Недвижимость'], 'cc-elec': ['Электроника'],
  'cc-work': ['Работа','Работы'], 'cc-serv': ['Услуги'], 'cc-taxi': ['Такси','Межгород'],
  'cc-pet': ['Животные','Питомцы'], 'cc-kids': ['Детское'], 'cc-sport': ['Спорт'],
  'cc-furn': ['Мебель'], 'cc-food': ['Продукты'], 'cc-free': ['Отдам даром']
};
```
- Объявления из Supabase помечаются `fromDb: true`
- `updateCatCounts()` считает реальные объявления по каждой категории

## Мультиязычность
- 3 языка: ru / kz / en
- Переключатель в navbar (дропдаун)
- Все тексты в объекте `LANGS`
- FAQ отдельно в `FAQ_DATA`
- `currentLang` — глобальная переменная, сохраняется в localStorage
